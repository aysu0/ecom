Star Toys and Kitchen Django E-commerce Project

Star Toys and Kitchen is a Django-based e-commerce platform for buying toys and kitchen products. It allows users to browse products, add them to their cart, and make purchases securely.

Installation
Clone the repository: git clone https://github.com/aysu0/ecom.git
Set up database: python manage.py migrate
Create a superuser: python manage.py createsuperuser
Run the development server: python manage.py runserver

Configuration
Create a virtual environment through python -m venv env
Activate virtual environment through source env/bin/activate
Download Django, Crispyforms, Stripe, Pillow

Usage
Access the admin panel at http://localhost:8000/admin to add products and manage orders.
Browse the website at http://localhost:8000 to view products, register, and make purchases.
